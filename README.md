"# Gestalt-MNIST" 

Gestalt MNIST is generally to add perturbations that are perceptible to human eyes, yet human would still classify the image as the original label.

In MNIST and FashionMNIST folder, you can find several Gestalt MNIST and FashionMNIST dataset in 'test.pt', the storage is the same as the one you downloaded using pytorch.

If you want to use the dataset already created, you can refer to 'gestalt_mnist.py'.

If you would like to create your own Gestalt dataset, refer to 'gestalt_transformation.py'.
